# App Web

Primeira experiencia web autenticada do usuario final da Ciclorota.

## O que ja existe

- login por email e senha usando a API
- restauracao simples de sessao com refresh token
- leitura de `/auth/me`, `/me/profile` e `/me/progress`
- atualizacao de perfil e emissao de certificado

## Ambiente

```env
VITE_API_URL=http://127.0.0.1:3333
```
